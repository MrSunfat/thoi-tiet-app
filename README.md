# thoi-tiet-app

## Công nghệ:
- React Native
- Expo
- React Navigation
- Firebase
- React Native Maps

## Chức năng:
- Tra thời tiết của các thành phố trên thế giới
- Lưu lịch sử tìm kiếm (3 thành phố gần đây)
